TANDEE TRAVEL TAP GPS v2

What it does:
- Uses browser GPS or DuBois test mode.
- Searches live OpenStreetMap/Overpass data for nearby gas, coffee, food, scenic stops, parks, playgrounds, churches/nonprofits.
- Speaks answers using browser speech synthesis.
- Opens selected results in Google Maps or OpenStreetMap.

Important:
- GPS works best after uploading to HTTPS, such as Netlify.
- OpenStreetMap data may be incomplete in rural areas.
- No API key required.
